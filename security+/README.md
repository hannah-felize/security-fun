# CompTIA Security+

Overview for the CompTIA Security+ (SY0-601) certification exam.

## Exam Domains

- Attacks, Threats, and Vulnerabilities
- Architecture and Design
- Implementation
- Operations and Incident Response
- Governance, Risk, and Compliance
